# coronavirus-stats
Simple and user friendly Coronavirus stats tracker.<br/>
You can follow up to 10 countries and keep track of their Coronavirus stats.

## DOWNLOAD
Because of the GitHub file upload limit (100 files) country flags are not included.
You can download them (optional) from <a href=https://www.mediafire.com/file/kbgedcsmg5gur4u/>MediaFire</a>.<br/>
Extract the Flags folder from the RAR and put it in the same directory as the source code (covid.py).

## REQUIREMENTS
The source code requires the following modules:
- tkinter
- PIL
- BS4 and requests
- pycountry
- pickle
- codecs
- threading
- os
